=======
Rick and Morty App 🌌

Esta es una aplicación web interactiva desarrollada con React y Vite, que consume la API oficial de Rick and Morty. Permite a los fans de la serie explorar personajes, filtrar por género o estado, y guardar sus favoritos.



🚀 Características principales
Exploración de personajes: Navegación por páginas de todos los personajes de la serie.

Filtros avanzados: Búsqueda por nombre y filtrado dinámico por género y estado (alive, dead, unknown).

Sistema de favoritos: Guarda tus personajes preferidos para consultarlos cuando quieras.

Detalle del personaje: Accede a información detallada de cada personaje.

Diseño responsivo: Adaptado para una experiencia fluida tanto en escritorio como en dispositivos móviles.

🛠 Tecnologías utilizadas
React: Biblioteca principal para la interfaz de usuario.

Vite: Herramienta de compilación rápida para un desarrollo eficiente.

React Router: Para la navegación entre rutas (/, /favoritos, /character/:id).

Zustand: Gestión de estado global para manejar los filtros y el paginado.

Lucide React: Biblioteca de iconos para una interfaz moderna.

CSS Modules: Para un estilo modular y mantenible.

📦 Instalación y ejecución local

Ejecuta el entorno de desarrollo:

⚙️ Estructura del Proyecto
